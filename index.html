Sync settings

Your journal is stored as a private GitHub Gist. Enter a personal access token with gist scope. Each device where you paste the same token + gist ID will stay in sync.

  1. Open github.com/settings/tokens → Generate new token (fine-grained).
  2. Set an expiration and give it Account permissions → Gists → Read and write.
  3. Copy the token (starts with github_pat_…) and paste below.
  4. Click Create new gist on the first device. On your next device, paste the same token and the Gist ID that appears.
