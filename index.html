Sync settings

Your journal is stored as a private GitHub Gist. Enter a personal access token with gist scope. Each device where you paste the same token + gist ID will stay in sync.

  1. Open github.com/settings/tokens → Generate new token (fine-grained).
  2. Set an expiration and give it Account permissions → Gists → Read and write.
  3. Copy the token (starts with github_pat_…) and paste below.
  4. Click Create new gist on the first device. On your next device, paste the same token and the Gist ID that appears.

Behavioral Activation · Build · Become

Hi. This is your daily container for the simple, durable thing: do small actions that matter, even when you don't feel like it. Feelings follow action — not the other way around.

Five ideas live inside:

  • Anchors with build-up steps. Big things start tiny. Lay out the clothes. Take the one breath. Each step counts.
  • Body check-ins (interoception). Two minutes a day to feel what's actually here.
  • Micro-practices. 30–90 seconds, rotating daily — for neuroplasticity, presence, or your edge.
  • Fear → fun reframe. Same body, different label. Confidence comes from showing up prepared.
  • Be where your feet are. Eileen Gu mindset: love the process, smile under pressure, treat fear as excitement.

You can start anywhere. The smallest step still counts.
